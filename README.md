# alchemy

Lightweight ECS-based game engine written in C++ on top of Vulkan.

## Setup

Install the following dependencies:

-   [Vulkan SDK](https://vulkan.lunarg.com/sdk/home)
-   [libjpeg-turbo](https://libjpeg-turbo.org/)
